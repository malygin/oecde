В 2008 году Саратовский Государственный университет начал реализовывать систему дистанционного обучения, которая делает возможным получение образования независимо от места проживания и удаленности от вуза.

Система дистанционного образования

  • для тех, кто хочет получить высшее, второе высшее образование;
  • для тех, кто удален территориально от нашего вуза;
  • для тех, кто проводит большую часть времени дома;
  • для тех, кто бережно относится к своему времени;
  • для тех, у кого Интернет является «естественной средой обитания»;
  • для тех, кто хочет найти единомышленников в социально-образовательной сети СГУ;
  • для тех, кто желает обучаться сразу в нескольких местах;
  • для тех, у кого жесткий график работы.
